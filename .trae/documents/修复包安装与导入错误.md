# 修复包结构与启动方式

我们将把项目配置为一个可安装的 Python 包，从而解决相对导入错误，并启用 CLI 命令。

## 1. 配置打包 (Packaging)

修改 `pyproject.toml`，添加构建系统配置，告诉 `uv` 这是一个标准的 Python 项目。

*   添加 `[build-system]` 部分 (使用 `hatchling` 或 `setuptools`，这里推荐 `hatchling` 配合 `uv` 很顺手)。
*   确保 `[project.scripts]` 配置正确。

## 2. 安装项目 (Install)

运行 `uv sync` (或者 `uv pip install -e .`)。这会把 `src/codesoul` 链接到虚拟环境中。
这样，Python 就能在任何地方通过 `import codesoul` 找到你的代码，相对导入也就正常了。

## 3. 验证 (Verify)

运行 `uv run codesoul --help` 确认命令已生效。
然后运行 `uv run codesoul start .` 启动 TUI。

不需要重命名外层文件夹 `13`，只要内部包结构正确即可。
