# GitHub 发布准备与推送

我们将完善项目文件，并将其推送到 GitHub 仓库 `zylar06/codesoul`。

## 1. 文件完善
*   **生成 requirements.txt**: 运行 `uv pip compile pyproject.toml -o requirements.txt`，方便非 uv 用户。
*   **添加 LICENSE**: 创建 `LICENSE` 文件（MIT 协议），内容与 GitHub 上的一致。
*   **更新 README.md**: 
    *   添加 `uv tool install` 的安装说明。
    *   更新 Shields 徽章链接。

## 2. Git 初始化与推送
*   初始化本地 Git 仓库：`git init`。
*   添加远程仓库：`git remote add origin git@github.com:zylar06/codesoul.git`。
*   **注意**: 这是一个新仓库，我们直接把所有文件 Add 并 Commit。
*   **推送**: `git push -u origin main` (或者 master，取决于你的默认分支)。

## 3. 待办事项
*   请确认你的本地环境已经配置好了 SSH Key，并且有权限推送到 `zylar06/codesoul`。
